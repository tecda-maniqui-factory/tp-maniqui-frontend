      Hook de control (Controller) para el formulario de ensamblaje (AssemblyForm).

      Se encarga de cargar los modelos disponibles en el catálogo (Modelo), gestionar el estado de selección de modelo y número de serie del nuevo maniquí, y orquestar el envío de la petición de ensamblaje mediante el servicio productionService.ensamblarManiqui.

      Se integra con los hooks useAuth y useNotify.

      Parameters

      • OptionalonSuccess: () => void

        Callback opcional ejecutado al concretarse con éxito el ensamblaje en el backend.

          };

      Estado local, métodos de cambio y envío, y utilidad de traducción.

      Example

      import { useAssemblyController } from './hooks/useAssemblyController';

      const MyForm = () => {
        const { modelos, selectedModelo, numeroSerie, handlers } = useAssemblyController(() => console.log('Éxito'));
        return (
          <form onSubmit={handlers.handleSubmit}>
            <input value={numeroSerie} onChange={handlers.handleSerieChange} />
          </form>
        );
      };
