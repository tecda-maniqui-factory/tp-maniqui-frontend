    Function useAuth

    • useAuth(): AuthContextType

      Hook personalizado para acceder rápidamente al contexto de autenticación global.

      Encapsula el consumo de AuthContext y verifica que se encuentre en el árbol de componentes dentro de un AuthProvider.

      Returns AuthContextType

      El estado y acciones de autenticación provistos por AuthContextType.

      Example

      import { useAuth } from '@/hooks/useAuth';

      const Dashboard = () => {
        const { user, logout } = useAuth();
        return <button onClick={logout}>Salir de {user?.username}</button>;
      };

      Throws

      Si el hook se invoca fuera del árbol de componentes envuelto por AuthProvider.
