    Servicio encargado de realizar las consultas a la API comercial del backend.

    Se consume principalmente dentro de useSalesController.

    Type Declaration

    • createCliente: (
          token: string,
          data: { cuit_cuil: string; email: string; nombre: string },
      ) => Promise<Cliente>

      Da de alta un nuevo cliente.

      Throws

      Si la sesión expira o si los datos (como CUIT) no pasan las validaciones del backend.

    • createVenta: (
          token: string,
          data: {
              cliente_id: number;

              metodo_pago: string;
              moneda: string;
          },
      ) => Promise<Venta>

      Registra una nueva venta de maniquíes en el sistema.

      Throws

      Si la sesión expira, si el cliente no existe, o si algún maniquí seleccionado ya no está disponible.

    • getClientes: (token: string) => Promise<Cliente[]>

      Obtiene la lista completa de clientes registrados.

      Throws

      Si la sesión ha expirado (auth.error.session_expired) o si ocurre un fallo de red.

    • getManiquiesDisponibles: (token: string) => Promise<Maniqui[]>

      Obtiene la lista de maniquíes que se encuentran listos para la venta.

      Throws

      Si la sesión expira o falla el servicio de red.

    • getVentaById: (token: string, id: number) => Promise<VentaDetalle>

      Recupera el detalle completo de una venta específica mediante su ID único.

      Throws

      Si la sesión expira o no se encuentra la venta solicitada.

    • getVentas: (token: string) => Promise<VentaRaw[]>

      Obtiene el listado histórico de todas las ventas del sistema.

      Throws

      Si la sesión expira o si falla la petición de carga.
